## Linux
* **Architecture**: x86 / x86_64 / arm64 / ppc64le / s390x
* **C/C++ Compiler**: GCC 4.8+ or Clang
* Tools:
    * GNU Make or [CMake](https://cmake.org/download/) 3.14.5+

## macOS
* **Architecture**: x86_64 / arm64 (Apple Silicon)
* **OS**: macOS 10.12+
* **C/C++ Compiler**: Apple XCode Clang
* Tools:
    * GNU Make or [CMake](https://cmake.org/download/) 3.14.5+
